# 🏨 Hotel Booking Analytics

## 📌 Business Problem
Hotels lose revenue through last-minute cancellations and poor demand forecasting.
This project analyses 119,000+ booking records to uncover cancellation patterns,
seasonal demand trends, and revenue optimisation opportunities.

## 🎯 Key Questions Answered
- What is the cancellation rate and what drives it?
- Which customer segments are most valuable?
- What booking channels generate the most revenue?
- How does lead time affect cancellation probability?

## 🛠 Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (data extraction and validation)
- Tableau (final dashboard)
